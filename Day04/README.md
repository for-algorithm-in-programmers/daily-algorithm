## 📚 Summary

- **난이도**: Lv2
- **주요 키워드**: Hash
- **핵심 아이디어**: map에 종류별 옷의 개수를 담고 경우의 수를 구한다.

## 🧠 리뷰 포인트

- 의상 종류별로 `Map`을 사용하여 의상을 분류함으로써 조합의 수를 계산하는 방식이 사용
- 각 의상 종류마다 선택하지 않는 경우를 포함하여 `(의상 수 + 1)`로 계산하고, 최종적으로 아무것도 입지 않는 경우를 제외하기 위해 `1`을 해주는 로직이 구현
